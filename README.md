# Simple Interest Calculator

This repository contains a shell script to calculate **Simple Interest** using user input.

## Description
The script takes three inputs from the user:
- Principal amount
- Rate of interest
- Time period

It then calculates and displays the Simple Interest.

## Formula Used
Simple Interest = (Principal × Rate × Time) / 100

## How to Run
1. Clone the repository
2. Open terminal in the project directory
3. Run the script:
   ```bash
   bash simple-interest.sh
