# Contributing to Simple interest Calculator

First off, thank you for considering contributing! To maintain a clean and organized project, please follow these steps:

 The Contribution Workflow
1. **Fork**https://github.com/B123-crypto/intrest-calculator
2. **Clone** Simple intrest calculator
3. **Create a Branch** for your specific feature or bug fix:
   `git checkout -b feature/your-feature-name`
4. **Commit** your changes with a descriptive message:
   `git commit -m "feat: add amazing new feature"`
5. **Push** to your fork:
   `git push origin feature/your-feature-name`
6. **Open a Pull Request (PR)** against the main repository for review.

### ⚖️ Guidelines
- Ensure your code follows the project's style.
- Be respectful and follow our Code of Conduct.
